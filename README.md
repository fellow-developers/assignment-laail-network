# laail-networks

- Pre-requisites
  - Node.js
  - NPM
  - Git



## Description

- Clone the project
```bash
git clone git@github.com:fellow-developers/assignment-laail-network.git
```

- Change directory to project
```bash
cd assignment-laail-network
```

- Install dependencies
```bash
npm install
```

- Build project
```bash
npm run build
```

- Run project
```bash
npm start
```

- Open browser and go to http://localhost:3000
